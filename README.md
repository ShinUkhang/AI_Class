# 🧠 NeuralFlow Visualizer - 인공신경망 단계별 시각화 시뮬레이터

이 프로젝트는 딥러닝 입문자들의 이해를 돕기 위해 **순전파(Forward Propagation) ➔ 오차 계산(Loss Calculation) ➔ 역전파(Backpropagation) ➔ 가중치 업데이트(Weight Update)**의 전 과정을 단계별 애니메이션과 실시간 연산 로그로 보여주는 프론트엔드 웹 기반 교육 도구입니다.

서버 없이 브라우저 환경에서 단독으로 작동하며, **GitHub Pages**에 업로드하여 즉시 고유 웹 링크로 배포가 가능하도록 설계되어 있습니다.

## 🚀 주요 기능 및 핵심 특징

1. **순전파 단계 시각화 (Forward Propagation)**
   - 데이터 샘플 패킷이 입력 노드에서 은닉 노드를 거쳐 출력 노드로 향하는 흐름을 빛나는 파티클 애니메이션으로 연출.
   - 선 위에 표시되는 가중치($W$)와 노드 내부에 실시간 변화하는 활성화 결과값 출력.
2. **오차 계산 단계 (Loss Calculation)**
   - 실제 타깃값(Target)과 신경망의 예측 결과값(Output)을 명확하게 비교.
   - 평균제곱오차(MSE)를 변형한 손실 함수의 수식과 함께 현재 손실도 그래프 게이지 및 텍스트를 실시간으로 변환.
3. **역전파 및 그래디언트 전파 (Backpropagation)**
   - 연산의 흐름이 반대로 역전되어 출력 노드에서 입력단으로 파티클이 역행하는 시각 효과 구현.
   - 편미분 체인 룰(Chain Rule)에 의해 가중치 오차 기울기($\delta$)가 전파되는 공식을 하단 로그에 실시간 매핑.
4. **대시보드 컨트롤 및 에포크 제한 제어**
   - 10~200회 범위에서 최대 학습 횟수(Max Epoch)를 슬라이더 바를 통해 손쉽게 변경 가능.
   - 속도 컨트롤러(느림/보통/빠름), 일시정지, 단계별 한 스텝씩 넘겨보기(Step by Step) 모드 지원.
   - 실제 비선형 분류 문제인 **XOR 연산 데이터셋**을 사용해 학습을 반복할수록 Loss 값이 0으로 하향 수렴해가는 기계학습 모델의 작동 원리를 입증.

## 📦 파일 구성
- `index.html`: 디자인 스킨(CSS) 및 신경망 연산 코어 엔진(JavaScript Canvas Engine)이 모두 통합된 완전 독립형 웹 빌드 파일.

## 🛠️ GitHub 저장소 생성 및 고유 웹 링크 배포(GitHub Pages) 방법

이 자료를 깃허브에 공유하고 실시간 웹 사이트로 배포하는 순서는 다음과 같습니다.

1. **GitHub 저장소 만들기:**
   - 개인 GitHub 계정에 접속한 후 `New repository` 버튼을 클릭합니다.
   - 저장소 이름(Repository Name)을 입력합니다. (예: `neural-network-visualizer`)
   - 공개 범위를 **Public**으로 설정하고 생성합니다.

2. **소스 파일 업로드:**
   - 생성된 저장소에 생성해 드린 `index.html` 파일을 업로드(Add file -> Upload files)하고 커밋(Commit)합니다.

3. **GitHub Pages 서비스 활성화:**
   - 저장소 상단 메뉴의 **Settings** (설정) 톱니바퀴 탭으로 들어갑니다.
   - 좌측 사이드바 메뉴 중 **Code and automation** 하위의 **Pages** 메뉴를 선택합니다.
   - **Build and deployment** 섹션의 Source 항목이 `Deploy from a branch`로 되어있는지 확인합니다.
   - Branch 설정을 `None`에서 **`main`** (또는 `master`)으로 변경하고 옆의 폴더 경로는 `/ (root)`로 둔 채 **Save** 버튼을 누릅니다.

4. **배포 확인:**
   - 약 1~2분 뒤 페이지를 새로고침하면 상단에 **"Your site is live at..."** 문구와 함께 고유 웹 사이트 주소(`https://사용자이름.github.io/저장소이름/`)가 나타납니다. 해당 링크를 통해 언제 어디서든 접속하여 시뮬레이션을 실행해 볼 수 있습니다!
